Inbox

The open source email toolkit.

Say goodbye to IMAP.

Inbox is a modern platform for email.

Easy to use

Inbox makes it simple and quick to write scripts that process or analyze your email data.

Backwards compatible

Use your existing GMail address and mail archive. No need change clients or providers.
Yahoo and iCloud coming soon!

No more MIME

Forget about old protocols and obscure formats. Inbox is built for this century. That means JSON, Unicode, stateless APIs, and proper unique IDs.

Open source

Inbox is open source free software, so you can run it on your own hardware. See the code on GitHub, or download a VM image or Docker container.
